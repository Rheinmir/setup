# Welcome to Memos 👋

An open-source, self-hosted note-taking tool for people who think in fragments. Capture quickly, organize lightly, own everything.

> Most apps treat notes like documents. Memos treats them like thoughts — short, timestamped, searchable.

## Try it right now

- [x] Open this memo
- [ ] React with 🎉 below
- [ ] Scroll the timeline to see what others have written
- [ ] Write your own first memo

## What you can do here

| Feature | Example |
|---------|---------|
| **Markdown** | Headings, **bold**, *italic*, `code`, ~~strikethrough~~ |
| **Tags** | Type `#anything` and it becomes a filter |
| **Task lists** | `- [ ]` checkboxes that toggle inline |
| **Code blocks** | Fenced blocks with syntax highlighting |
| **Tables** | Pipes and dashes — yes, this one |
| **Attachments** | Drag images, videos, or files right in |
| **Location** | Geotag a memo to where you wrote it |
| **Relations** | Link memos together as references or replies |

## Self-host in one command

```bash
docker run -d -p 5230:5230 -v ~/.memos:/var/opt/memos neosmemo/memos:stable
```

Then open `http://localhost:5230` and start writing.
