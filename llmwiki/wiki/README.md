# llmwiki/wiki — khuôn wiki per-project (template)

Đây là **khuôn wiki** mà mỗi project dùng overstack sẽ có. **Wiki RIÊNG của repo framework**
(ADR, concepts harness/fdk, decisions…) nằm ở **`fdk/wiki/`** — "the kit" (xem ADR-008).

Thư mục này trong repo gốc chỉ giữ **1 file demo** (`concepts/example-concept.md`) để CI
vẫn dogfood được luật wiki trên cả hai root.
