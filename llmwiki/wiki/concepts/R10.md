# R10 — docs-gate

Cứ mỗi N prompt (mặc định 5), nếu N request gần nhất chưa dùng `/docs-site-macos` hay `/orca-workflow` thì nhắc bổ sung tài liệu. Hook `user_prompt_submit.py` inject directive vào prompt mốc-N — báo cáo, không gate.

## Cấu hình

- Hook: `llmwiki/.claude/hooks/user_prompt_submit.py`
- Trạng thái đếm: `.claude/audit/.docs-gate.json`
- Nhịp: `LLMWIKI_DOCS_GATE_EVERY` (default 5)
- Enforce at: `[session]` — UserPromptSubmit
