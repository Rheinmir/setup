# sources/adr/ — Architecture Decision Records

Mỗi quyết định kiến trúc/kỹ thuật quan trọng = **1 file** `ADR-NNN-slug.md` (`type: decision`).

Cấu trúc chuẩn: `## Status` · `## Context` · `## Decision` · `## Consequences` · `## Origin`.
Đánh số tăng dần; **không sửa ADR cũ** — nếu đổi ý thì viết ADR mới *supersede* cái cũ.

Dùng `_template.md` trong thư mục này làm khung. ADR là first-class trong khuôn wiki
per-project (validators R5/R9/R2/R3 đều chấp nhận `sources/adr/`).
