# Architecture Decision Records

> ADR ghi lại các quyết định kiến trúc quan trọng, lý do chọn giải pháp.

## Template
```markdown
# ADR-<số>: <Tiêu đề>

## Status
[Proposed | Accepted | Deprecated | Superseded]

## Context
Vấn đề cần giải quyết, các ràng buộc.

## Decision
Giải pháp được chọn.

## Consequences
Tác động tích cực/tiêu cực.
```
