01 · ĐỊNH NGHĨA

Framework hay Template? — Là cả hai

Câu hỏi đúng không phải "chọn cái nào" mà là "ở tầng nào". Cơ chế là một framework; cách phân phối là một template. Tên gọi gọn nhất cho cả gói: llmwiki.

🧩 Là FRAMEWORK (bản chất)

  • Áp đặt luật vận hành: không ghi raw/, mọi entry có ## Origin, code commit xong mới ghi wiki.
  • Định nghĩa 4 vòng lặp (wiki / dev / orchestrate / utils) và khi nào agent tự gọi skill nào.
  • Có runtime: knowledge graph (graph.db), memory, RTK token-proxy, Caveman compression.

📦 Là TEMPLATE (phân phối)

  • Copy thư mục vào project mới rồi feed 4 file prompt — agent tự scaffold.
  • .template-manifest.json + sync-template giữ template gốc và project con đồng bộ.
  • Cài qua npx skills add … --global — skill dùng lại cho mọi repo.
02 · KIẾN TRÚC

4 vòng lặp quay quanh một knowledge base

Mọi skill thuộc về một "loop". Tất cả đổ tri thức vào — hoặc rút tri thức ra từ — wiki/. Kéo từng ô để xem quan hệ rõ hơn.

Luồng tri thức (wiki-loop)

raw/ → ingest distill thành trang wiki → query tổng hợp khi cần → lint giữ wiki sạch (orphan, link hỏng, stale). Agent chỉ đọc raw/, không bao giờ ghi.

Luồng phát triển (dev-loop)

Mọi feature: propose (dừng chờ duyệt) → impact-check → safe-change → verify-before-commit. Wiki chỉ được ghi sau khi code đã commit.

Ý nghĩa màu — xanh dương: nuôi & truy vấn knowledge base · xanh ngọc: cổng an toàn quanh mỗi thay đổi code · chàm: khởi tạo dự án & map codebase · lục: điều phối nhiều agent qua Orca · cam: nén ngữ cảnh để tiết kiệm token · hồng: tiện ích xuất bản (HTML, markdown, đồng bộ template).

ingest
Distill file mới trong raw/ thành trang wiki.
query
Tổng hợp câu trả lời từ wiki; lưu insight mới.

propose
Lập kế hoạch feature, tạo draft, dừng chờ duyệt.
impact-check
Map mọi caller/dependent trước khi sửa code chung.
safe-change
Sửa code dùng chung mà không phá caller cũ.
verify-before-commit
Gate commit: typecheck, lint, smoke, promote draft.
new-project-setup
Deploy llmwiki vào project mới từ đầu.
join-project
Orient nhanh vào dự án đã có llmwiki (read-only).
onboard-codebase

orca-workflow
Hằng ngày: propose → gate → dispatch với Orca.
orca-onboard
Onboard song song nhiều agent Orca.
orca-dispatch-reference
