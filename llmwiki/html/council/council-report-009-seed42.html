Bộ harass tối thiểu-mà-đủ, mỗi đòn một tiêu chí đúng/sai sắc:

  1. imports re-export: cli/__init__.py re-export resolver.core qua from ..resolver import *, không gọi trực tiếp — pass nếu nối cli/__init__→resolver/core bằng imports mà KHÔNG bịa touches; fail nếu bỏ sót hoặc gán nhầm wikilink.
  2. touches vs derives-from rename: commit đổi tên lockfile/v1.py→lockfile/v2.py kèm sửa nội dung — pass nếu v2 nhận derives-from:v1 song song touches của commit, fail nếu gộp một mối hoặc mất dấu tổ tiên.
  3. supersedes vs contradicts gần-giống: ADR-B supersedes ADR-A (thay trật tự), ADR-C contradicts ADR-A (mâu thuẫn cùng tồn tại) — pass nếu giữ đúng hai hướng, fail nếu quy về một loại.
  4. depends-on vs implements trừu tượng→cụ thể: registry_client depends-on cache-interface còn redis_cache.py implements cache-interface — pass nếu không hoán đổi chiều, fail nếu đảo.
  5. wikilink nhiễu: [[cache]] trong khối code và [[cache]] trong văn xuôi — pass nếu chỉ link cái ngoài code.
  6. code↔code imports vòng: resolver import lockfile, lockfile import resolver — pass nếu ghi hai cạnh imports không treo/nhân đôi vô hạn.
  7. âm tính cám dỗ: hai file cùng tên hàm resolve() nhưng không import nhau — pass nếu KHÔNG bịa vector. Chấm: mỗi đòn nhị phân đúng/sai trên cạnh và nhãn loại; qua khi ≥6/7 và tuyệt đối không dương-tính-giả ở đòn 5 và 7.
